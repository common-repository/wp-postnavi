== Description ==

**WP PostNavi**

WordPress Inpage Post Navigation Plugin is a smart WordPress plugin which allows smooth navigation on websites by building link around the 

headers on different pages. This plugin is designed to provide extra convenience to the site visitors who can simply click on the inpage post 

navigation and reach the desired page. 
Inpage Post Navigation WordPress Plugin is built to support wordpress sites with extra navigational features like Post Link Text Prefix that 

develops link to the header on main page. Link Color gives the choice to pick up colors. With this plugin you can also keep the position of 

post navigation on left or right side of your page and also keep it in fixed position with the PostNavi Position features.

= Plugin Features =
* Inpage navigation for blog/archive pages
* Smooth scrolling effect
* Navigation dock position Left/Right
* Post links prefix text
* Set top position option
* Link color option

== Installation ==
1. Upload 'wp-postnavi' to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Activate the plugin via the 'Plugins' menu in WordPress, you'll automatically redirected to plugin admin panel.
3. Plugin settings are located in 'WP PostNavi'.
